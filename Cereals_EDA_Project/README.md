# 🥣 Cereal Data Analysis (EDA Project)
## 🌏 Overview
This project analysis a cereal dataset to understand manufacturer wise ratings and identify top performing brands using python

## 🛠️  Tools used
Python,Pandas,Matplotlib,Seaborn

##  📊 Key Insights
Manufacturers show variation in ratings , where some brands consistently performing better, indicating higher quality and preference.
