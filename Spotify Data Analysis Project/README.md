# Spotify top 50 Songs Analysis
## Project Overview
Analysis of Spotify Top 50 songs dataset to find top artist,genre,and music trends using Python.

## Tools 
Python, Pandas, Jupyter Notebook

## Key Insights
The top artist has maximum track in the dataset
-This indicates that he has strong catch in a particular genre.
-This also suggest dominance in music sector.
